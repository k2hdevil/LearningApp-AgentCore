## Course Objectives

Upon completing this course, you will be able to:

1. **Build production-ready agents using AWS services.**
2. **Deploy agents using Amazon Bedrock AgentCore services.**
3. **Integrate multiple tools and services for agent workflows through Amazon Bedrock AgentCore Gateway.**
4. **Configure credential management for agent authentication and authorization.**
5. **Apply monitoring and observability patterns to optimize agent performance.**

### Time Adjustment Strategy

**When running behind (reduce in priority order)**:
1. Discussion topics → Ask question only and move on (saves ~5 min per module)
2. Knowledge check questions → Solve 1 only, recommend self-study for the rest
3. Supplementary content (Payments, Registry) → Brief one-line mention
4. Lab review → Reduce to 15 min

**When ahead of schedule**:
- Deep-dive into discussion topics (agent design activity)
- Supplementary content demo (Create Harness in real-time with CLI)
- Extended Q&A
